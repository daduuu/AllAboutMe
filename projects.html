Discrete Calculator

Discrete Calculator is a Java-based calculator which is capable of complex computations. Not only can it perform the four basic functions, it can perform them with the addition of variables, because of the primitive CAS system implementation. It can also show basic graph data and calculate values of certain functions. It also has the capability to create and manipulate vectors and to load basic shapes. This is a WIP
